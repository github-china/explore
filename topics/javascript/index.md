JavaScript(简称JS) 是一种非常流行的解释型脚本编程语言、单线程。
虽然作为 Web 网页中的脚本语言被人所熟知，但是它也被用到了很多非浏览器环境中，例如 Node.js、Apache CouchDB、Adobe Acrobat 等。
