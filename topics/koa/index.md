Koa 是由 Express 团队设计的新 Web 框架，旨在为 Web 应用程序和 API 提供更小、更具表现力和更强大的基础。
通过利用异步函数，Koa 允许您抛弃回调并大大增强错误处理能力。Koa 的核心中不捆绑任何中间件，它提供了一套优雅的方法，使编写服务器变得快速而愉快。
