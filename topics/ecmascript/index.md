**ECMAScript** 是由Ecma国际制定的 JavaScript 标准规范。该标准每年都会发布新版本。ECMAScript 通常是用于运行在浏览器上的客户端脚本语言，如今越来越多地用于使用 Node.js 和其他运行时环境编写服务器端程序。
