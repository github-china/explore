低代码是（Low Code）一种软件开发方法，使用可视化的方式进行软件开发。低代码开发平台，旨在实现快速应用程序开发。低代码平台不需要开发人员编写代码，而是提供图形用户界面 (GUI)，用户可以使用该界面配置和自定义应用程序。低代码平台还可以提供可视化编程语言，让用户无需编写代码即可创建应用程序。
