Edge 是微软基于 Chromium 开发的跨平台浏览器，是Windows 10后的默认浏览器。

2015 年首次发布适用于 Windows 10，2017 年发布适用于 Android 和 iOS，
2019 年发布适用于 macOS，2020 年发布适用于 Linux，
可在 Windows 7、Windows Server 2008 R2 及更高版本上替代 Internet Explorer。
