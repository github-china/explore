Advent of Code 是一个编程挑战活动，每年的12月份举行，持续25天。

从每年的 12 月 1 日到 25 日，每天都会在其官方网站上发布一个新的编程谜题，总共 25 个。
这些谜题通常都基于一个有趣的故事背景，例如拯救圣诞节、帮助圣诞老人等，使参与者仿佛置身于一个充满节日氛围的编程世界中。
