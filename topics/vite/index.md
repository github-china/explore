Vite 是一款适用于现代前端开发的 JavaScript 开发服务器和打包器。
使用原生 ESM 文件 无需打包，具有丰富的内置功能(对TypeScipt、JSX、CSS等支持开箱即用) 和极快的模块热替换 (HMR)，从而增强了开发体验。
