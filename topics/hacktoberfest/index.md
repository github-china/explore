**Hacktoberfest** 是每年为期一个月的庆祝活动，旨在鼓励人们参与开源社区和贡献开源项目。
每年十月，开源维护者都会给予新贡献者额外的关注，指导开发人员完成他们在 GitHub 上的第一个拉取请求。
