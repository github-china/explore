NativeScript 是 Progress 开发的 JavaScript 原生移动框架。
NativeScript 帮助开发人员使用现有的 Web 技能（JavaScript、CSS、XML）和成熟的框架（Angular 和 Vue）构建真正的原生跨平台 iOS/Android 移动应用。
