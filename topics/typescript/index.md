TypeScript 是微软开发的一种开源编程语言，于 2012 年首次发布。它旨在为 JavaScript 添加类型安全性，
同时尽可能符合 ECMAScript 标准的语法和语义。

它是 JavaScript 编程语言的语法超集。可以编译为纯 JavaScript，这意味着它可以用于任何 JavaScript 环境。
