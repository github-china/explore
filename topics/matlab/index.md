MATLAB 是由 MathWorks 开发的一种高性能语言，主要用于技术计算、可视化和编程。
它使用 C、C++ 和 Java 编写，支持 Windows、macOS 和 Linux 操作系统。
MATLAB 通常被工程师和科学家用于工业界和学术界的数据分析、信号处理、优化以及许多其他类型的科学计算。
