Atom 是 GitHub 开发的一款现代开源文本编辑器。它设计为开箱即用且高度可定制。Atom 使用 Web 技术构建：可以使用 CSS 定制外观，可以使用 HTML 和 JavaScript 添加新功能。还有数千个社区创建的主题和插件可用。
